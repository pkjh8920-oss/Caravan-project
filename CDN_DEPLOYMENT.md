# ⚡ CDN 배포 전략 (Image Optimization)

## 목표
전 세계 사용자에게 카라반 이미지, CSS, JS 등 정적 자원을 빠르게 전송하여 로딩 속도를 최적화합니다.

## 주요 전략
1. **자산 저장소**: 로컬 디스크 저장 방식에서 **AWS S3** 객체 스토리지로 이관
2. **콘텐츠 전송**: **AWS CloudFront** 또는 **Cloudflare** 를 사용하여 엣지(Edge) 서버 캐싱 적용
3. **최적화**:
   - 업로드 시 이미지를 WebP 포맷으로 자동 압축
   - 목록 조회 시 Lazy Loading (지연 로딩) 적용

## 구현 계획
- **현재 (AS-IS)**: Express 서버의 `public/` 폴더에서 직접 서빙
- **향후 (TO-BE)**: `multer-s3` 라이브러리를 통해 S3 버킷으로 직접 업로드 구현
